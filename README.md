# 旅游可视化系统 python595

## 项目概述

旅游可视化系统是一款基于 Python 语言和 Flask 框架开发的 Web 应用，旨在通过数据可视化的方式为用户提供旅游相关的信息展示。

## 技术栈

- 后端：Python 3.x，Flask
- 数据库：SQLite
- 前端：HTML, CSS, JavaScript

## 功能模块

- 用户认证
  - 用户注册
  - 用户登录
  - 用户信息管理
- 旅游信息展示
  - 景点数据可视化
  - 旅游攻略分享
  - 行程规划工具
- 数据管理
  - 数据导入导出
  - 数据清洗与处理

## 系统角色

- 访客：浏览和搜索旅游信息
- 注册用户：拥有访客权限，并可发布和管理个人旅游内容

## 运行环境

- 操作系统：兼容 Linux、Windows 和 macOS
- Python 环境要求：Python 3.x
- 依赖管理：pip

## 部署步骤

1. 准备 Python 运行环境
2. 安装依赖包
   ```bash
   pip install -r requirements.txt
   ```
3. 配置数据库
4. 运行应用
   ```bash
   flask run
   ```

- **数据可视化**：通过图表直观展示旅游数据，便于用户理解与分析。
